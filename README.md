# O Projeto

Aplicativo voltado para administração de cadastro de médicos, clientes e consultas de uma clínica médica.


## Objetivos

* Desenvolvimento de uma API Rest
* CRUD (Create, Read, Update e Delete)
* Validações
* Paginação e ordenação

## Tecnologias

* Spring Boot 3
* Java 17
* Lombok
* MySQL / Flyway
* JPA / Hibernate
* Maven
* Insomnia
